# 音频资源目录

此目录存放游戏的音频资源文件。

## 所需音频文件

| 文件名 | 用途 | 格式 | 建议时长 |
|--------|------|------|----------|
| bgm.mp3 | 背景音乐 | MP3 | 60-120s 循环 |
| shoot.mp3 | 射击音效 | MP3 | 0.1-0.3s |
| player_hit.mp3 | 玩家受伤 | MP3 | 0.2-0.5s |
| enemy_hit.mp3 | 敌人被击中 | MP3 | 0.1-0.2s |
| enemy_die.mp3 | 敌人死亡 | MP3 | 0.3-0.6s |
| level_up.mp3 | 升级音效 | MP3 | 1-2s |
| exp_pickup.mp3 | 经验拾取 | MP3 | 0.1-0.2s |
| explosion.mp3 | 爆炸音效 | MP3 | 0.5-1s |

## 注意事项

1. 所有音频文件应为 MP3 格式以确保浏览器兼容性
2. BGM 建议为循环音乐，无缝衔接
3. 音效文件应尽量小，避免加载时间过长
4. 如果没有音频文件，游戏仍能正常运行（只是没有声音）
